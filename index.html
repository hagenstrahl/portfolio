Hagen Strahl

Hallo, mein Name ist Hagen Strahl. Schön, dass Sie mich gefunden haben.

Ich bin ein Software-Entwickler mit Spezialisierung für Web Frontend. Dabei intressiere ich mich besonders für JavaScript - wie es sich für einen Web Frontend Developer gehört.

Geboren wurde ich Ende der 80er im wunderschönenen Mecklenburg Vorpommern. Schon früh begeisterte ich mich für das Programmieren und auch während verschiedener Stationen in meinem Leben ließ es mich nie ganz los. Nach einer Ausbildung zum Fachinformatiker für Anwendungsentwicklung, vielen Jahren als System Administrator, einer angefangenen Ausbildung als Mediengestalter Digital/Print und zuletzt eine Zertifizierung bei Udacity habe ich nun mein Ziel erreicht: Ich bin ein Web Frontend Developer!

Web Frontend Developer in der Entwicklung

Erst vor Kurzem habe ich so richtig mit Web Frontend angefangen. Daher ist mir bewusst, dass ich noch am Anfang stehe und ich noch viel zu lernen habe. In dem Beruf kann man sowieso nie auslernen. Jedoch habe ich schon ein paar persönliche Erfolge erzielen können, die Sie unter den Referenzen sehen können.
